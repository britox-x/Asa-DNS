Bem-vindo ao servidor ASA-DNS

Este servidor foi configurado para fornecer serviços DNS e web para o domínio asa.br.

O servidor DNS está configurado com o Bind9 e o servidor web está rodando o Nginx. Este ambiente foi configurado usando Docker.

Para mais informações sobre o projeto, visite nosso repositório no GitHub:

Projeto ASA-DNS no GitHub

Para testar a resolução de DNS, você pode usar comandos como dig ou nslookup para verificar o funcionamento correto do servidor DNS.

Obrigado por usar o ASA-DNS!
